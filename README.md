# Flappy Bird Game

A simple Flappy Bird game developed using C# and .NET Windows Forms.

## Features
- Interactive GUI
- Smooth bird movement
- Gravity-based gameplay
- Randomly generated pipes
- Collision detection
- Score tracking
- Game Over and Restart functionality
- Simple and responsive UI

## Technologies Used
- C#
- .NET 8
- Windows Forms
- Visual Studio 2022

## How to Run
1. Clone the repository.
2. Open `FlappyBird.sln` in Visual Studio 2022.
3. Build and run the project.
